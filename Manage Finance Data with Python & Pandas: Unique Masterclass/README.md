## Course Overview
This masterclass teaches how to analyze stocks and manage financial data using Pandas, NumPy, Seaborn, and ChatGPT. You'll learn to create, analyze, and optimize indexes and portfolios, including CAPM, Alpha, and Beta calculations. The course includes a final capstone project where you step into a Financial Analyst role.

## What You'll Learn

### Pandas & Data Handling
- Import financial datasets from web sources, Excel, and CSV files
- Clean, aggregate, manipulate, and visualize financial data
- Switch between daily, weekly, monthly, and annual returns
- Understand log returns vs. simple returns
- Rolling statistics and moving averages

### Financial Analysis
- Calculate return, risk, correlation, and rolling statistics
- Create and analyze financial indexes (price-weighted, equal-weighted, value-weighted)
- Understand Price Return vs. Total Return
- Create, analyze, and optimize portfolios
- Calculate Sharpe Ratio, Beta, Alpha, systematic & unsystematic risk

### Modern Portfolio Theory & CAPM
- Modern Portfolio Theory (MPT)
- Capital Asset Pricing Model (CAPM)
- Capital Market Line (CML) & Security Market Line (SML)
- Risk diversification concepts
- Mean-Variance Optimization (MVO) and its real-world pitfalls

### Visualization & Technical Indicators
- Interactive price charts with technical indicators
- Volume, OHLC, Candlestick charts
- Simple Moving Averages (SMA)
- Bollinger Bands

### ChatGPT Integration
- Use ChatGPT for Pandas and data analytics

## Capstone Project
Step into a Financial Analyst/Advisor role and provide advice on a client's financial portfolio, applying all concepts learned throughout the course.

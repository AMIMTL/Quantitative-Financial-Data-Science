## Course Overview
This comprehensive bootcamp teaches Python coding from zero within a business, finance, and data science context. It covers theory at the level of the CFA curriculum while building practical Python skills with real-world examples.

## What You'll Learn

### Python & Data Science Stack
- Python from basics to advanced (Standard Library)
- NumPy and SciPy for numerical, financial, and scientific computing
- Pandas for tabular data processing (cleaning, merging, manipulating)
- statsmodels for regression analysis and ANOVA
- Matplotlib and Seaborn for data visualization
- User-defined functions for business applications

### Business & Finance
- Time Value of Money, Capital Budgeting
- Risk, Return & Correlation
- Monte Carlo Simulations
- Sharpe Ratio, Alpha, Beta, IRR, NPV, Yield-to-Maturity (YTM)
- Value-at-Risk (VaR), Portfolio Theory, Factor Models
- Mortgage Loans, Annuities, Retirement Planning

### Statistics
- Descriptive and inferential statistics
- Probability distributions (Normal, Student's t)
- Confidence intervals and hypothesis testing
- Bootstrapping method
- Normality of returns

### Regression Analysis
- Covariance and correlation
- Linear and multiple regression
- ANOVA and dummy variables
- Logistic regression
- Fama-French factor models

## Key Features
- No prior coding required – starts from absolute zero
- Theory aligned with CFA curriculum standards
- Avoids dubious practices (no LSTM stock price predictions)
- Downloadable Jupyter notebooks with thousands of lines of code
- Hundreds of coding exercises with hints and solutions
- PDF slides explaining key concepts

## Requirements
- Desktop computer (Windows, Mac, or Linux) for Anaconda
- Internet connection for video streaming
- High school math helpful but not mandatory

## Course Overview
This is the most comprehensive course on automated stock trading and equity investing [citation:1][citation:2]. It covers two core areas: **Automated Stock Trading for income generation** (algo trading, day trading) and **Automated ETF & Equity Portfolio Investing** for long-term wealth accumulation (passive, semi-active, and active investing) using Python and the Interactive Brokers (IBKR) API.

## What You'll Learn
- **Algorithmic Stock Trading** with Python and the Interactive Brokers (IBKR) API
- **Automated ETF & Equity Portfolio Investing** (passive, semi-active, and active)
- **Trading & Investment Strategies** with multiple tickers
- **Equity Portfolio Optimization**, management, and rebalancing
- **Backtesting** and implementation of trading strategies
- **Technical Analysis** and indicators
- **Equity Valuation Methods** (DDM and Multiples)
- **Fundamental Analysis**
- **Stock Indices** and Index Tracking/Replication
- **Performance Measurement, Benchmarking & Improvement** for equity portfolios
- **Loading and Analyzing Stock Data** from free web sources
- **API Trading** with Interactive Brokers
- **Asset-Pricing Models** (CAPM)
- **Black-Litterman Model** [citation:1][citation:2]

## Course Structure
The course is organized into 38 chapters covering:
1.  **Basics and Prerequisites:** Equity markets, Python/Jupyter installation, equity analysis with Python.
2.  **ETF Trading & Equity Portfolio Investing:** Building stock indices, ETF investing, portfolio optimization, CAPM, and the Black-Litterman model.
3.  **Algorithmic Stock Trading:** Trading strategies, backtesting, technical analysis with Python.
4.  **Advanced VIP Topics:** Equity valuation, advanced data sources, data streaming, and algorithmic day trading.
5.  **Appendix - Python Crash Course:** Covering Python basics, functions, NumPy, Pandas, Matplotlib, Seaborn, and Object-Oriented Programming (OOP) [citation:1][citation:2][citation:6].
